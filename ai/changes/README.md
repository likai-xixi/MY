# Change Records

Each Codex chat-driven feature change gets a folder named `CR-YYYY-MM-DD-HHMMSS-<slug>`.

Required files in each change record:

- `request.md`
- `impact.json`
- `plan.md`
- `changed-files.json`
- `verification.md`
- `handover.md`

Codex must create or update a change record before editing business code.
