# Session Log

Use this folder for session-level project memory.

Each session note records the goal, changed files, verification, risks, and the next entry point for the following Codex session.

Use sortable filenames:

`YYYY-MM-DD-<task-or-topic>.md`

Required headings:

- `## Task`
- `## Status`
- `## Goal`
- `## Changed Files`
- `## Commands`
- `## Verification`
- `## Risks`
- `## Next Entry Point`
